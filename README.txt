
-- SUMMARY --

The BTM Support HP Slideshow Simple module is the support module to the
BTM Features HP Slideshow Simple Features-generated module that eases the
deployment of a simple home page slideshow.

The BTM Support HP Slideshow Simple module simply makes sure that the Jquery
Cycle Plugin module, that is used by the BTM Features HP Slideshow Simple to
generate the slideshow, and the custom javascript file that calls the Cycle
function are included on the home page.


-- REQUIREMENTS --

None.


-- INSTALLATION --

* This module is dependent on the following modules:

  - Jquery Plugin - contains the cycle plugin that is used by this module

* Download the module

$ cd path/to/modules/contrib_local
$ git clone git@github.com:ShaunLaws/D7-BTM-Support-HP-Slideshow-Simple.git

* Download this dependent module to your site if it is not already installed

* Install as usual, see http://drupal.org/node/70151 for further information.
